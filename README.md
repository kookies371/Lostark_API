# 🎮 로스트아크 캐릭터 정보 조회 웹서비스

**로스트아크 캐릭터의 기본 정보, 장비, 카드를 웹에서 간편하게 조회하는 서비스**

캐릭터 이름 하나로 현재 보유한 장비, 착용 중인 카드, 계정의 모든 캐릭터 정보까지 한눈에 확인할 수 있습니다.

---

## 🚀 시작하기

### 1️⃣ 준비물

- 로스트아크 공식 개발자 포털에서 발급받은 **JWT 토큰** (API 키)
  - [로스트아크 개발자 포털](https://developer.game.onstove.com/) 방문
  - 계정으로 로그인 후 JWT 토큰 발급

### 2️⃣ 설정

프로젝트 루트에 `.env.local` 파일 생성:

```bash
LOSTARK_JWT=your_jwt_token_here
```

**⚠️ 보안 주의:**
- 이 파일은 git에 업로드되지 않습니다 (`.gitignore`에 제외됨)
- 토큰이 노출되면 개발자 포털에서 즉시 재발급하세요

### 3️⃣ 실행

프로젝트 루트에서 아래 명령어 실행:

```bash
# 백엔드 서버 시작 (FastAPI)
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

그 후 브라우저에서 **http://localhost:8000** 접속하면 웹 인터페이스가 표시됩니다.

## 📖 사용 방법

### 탭 1️⃣ : 계정 캐릭터

**캐릭터 이름 입력 → 검색** → 계정 내 모든 캐릭터를 서버별로 확인

- 캐릭터 이름
- 클래스 (직업)
- 레벨 & 아이템 레벨
- 소속 서버

### 탭 2️⃣ : 스펙 정보

**캐릭터 이름 입력 → 검색** → 장비와 카드 상세 정보 확인

#### 🛡️ 장비 정보
- 악세서리 (목걸이, 귀걸이, 반지)
- 각 아이템의 등급 (전설, 영웅, 희귀 등)
- 아이템 타입 및 이름

#### 📇 카드 정보
- **카드 그리드**: 6개 카드 한눈에 보기
  - 카드 이미지
  - 각성도 (진행바)
  - 슬롯 번호 표시

- **세트 효과**: 현재 카드 구성으로 얻는 효과
  - 활성화된 슬롯
  - 각 세트 효과의 설명 및 스텟 증가량

## 🎨 주요 기능

✅ **빠른 조회**
- 캐릭터 이름만 입력하면 바로 조회
- 첫 조회 후 결과가 로컬에 캐시되어 재조회 시 빠름

✅ **명확한 정보 표시**
- 등급별 색상 구분 (황금색 = 전설, 보라색 = 영웅, 파란색 = 희귀)
- 마우스를 올리면 효과 (hover) 확인 가능

✅ **반응형 디자인**
- 데스크톱, 태블릿, 모바일 모두 지원

✅ **서버별 그룹화**
- 캐릭터가 많으면 서버별로 자동 정렬
- 캐릭터 수가 많은 서버 우선 표시

## ❓ 자주 묻는 질문

**Q: "API 호출 실패" 에러가 발생합니다**
- A: `.env.local` 파일의 JWT 토큰이 정확한지 확인하세요
- 토큰이 만료되었다면 개발자 포털에서 새로 발급받으세요

**Q: 조회 속도가 느립니다**
- A: 첫 번째 조회는 로스트아크 공식 API에서 데이터를 받으므로 시간이 걸릴 수 있습니다
- 이후 같은 캐릭터를 조회하면 로컬 캐시에서 빠르게 로드됩니다

**Q: 서버가 안 켜져요**
- A: 다른 프로그램이 8000번 포트를 사용 중일 수 있습니다
  - 포트 변경: `--port 8080` 옵션으로 다른 포트 사용 가능
  - 예: `uvicorn app.main:app --reload --host 0.0.0.0 --port 8080`

**Q: 존재하지 않는 캐릭터를 조회하면?**
- A: "조회 실패" 또는 "캐릭터를 찾을 수 없음" 메시지가 표시됩니다
- 캐릭터 이름의 띄어쓰기나 철자를 다시 확인해보세요
